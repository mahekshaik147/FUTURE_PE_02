# AI Content Marketing using UGC Ads
## FUTURE_PE_02 – Future Interns Prompt Engineering Task

---

# 📌 Project Overview

This project focuses on generating AI-powered UGC (User Generated Content) advertisements for social media platforms using Prompt Engineering techniques.

The project demonstrates how AI tools can create authentic, conversion-focused ad content for real businesses.

---

# 🎯 Objective

The goal of this project is to:

- Generate UGC-style ad scripts
- Create scroll-stopping hooks
- Build platform-specific advertisements
- Use Problem → Solution → CTA frameworks
- Produce conversion-focused marketing content using AI
